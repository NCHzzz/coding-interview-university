# Study Dashboard

Use [[00 - Foundation Index]] as the single source of truth. Every learning section from `README.md`, including Additional and Optional content, now lives directly in `02 - Foundation`.

## Main Flow

1. Open [[00 - Foundation Index]].
2. Study sections in order.
3. Check off items inside each Foundation file.
4. Write your own summary in `My Notes`.
5. Create implementation notes in [[Implementation Index]] when a section asks you to implement something.
6. Create problem notes in [[Problem Index]] when a section asks you to practice questions.
7. Review progress in [[Weekly Review]].
